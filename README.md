# 💜 Romantic Apology Web Page

A beautiful and interactive **love proposal & apology website** made using HTML, CSS, and JavaScript.

This project is specially created for someone very special — ❤️

---

## ✨ Features

- 💬 Typing animation for romantic message
- 🎵 Background music with play/pause button
- ❤️ Floating animated hearts
- 🎆 Fireworks animation on "YES" click
- 😄 Funny "NO" button that runs away
- 📸 Memory photo section reveal
- 💖 Emotional apology message

## ⚙️ How to Use

1. Download or clone this repository
2. Add your files:
   - Replace `music.mp3` with your favorite romantic song 🎵
   - Add your photos as `photo1.jpg`, `photo2.jpg` 📸
3. Open `index.html` in your browser

---

## 🌐 How to Upload on GitHub

1. Go to GitHub
2. Click **New Repository**
3. Upload:
   - `index.html`
   - `music.mp3`
   - images
4. Click **Commit**

---

## 🚀 Enable GitHub Pages (IMPORTANT)

1. Go to **Settings**
2. Click **Pages**
3. Select:
   - Branch → `main`
4. Save ✅
